# ResumeIQ Product Notes

## Current Product Feedback

The website has improved from a basic ATS checker into a more complete product, but the next priority is product clarity and brand trust.

## What Is Working

- Clear core functionality: upload resume, add job description, get score.
- Keyword matching and missing skills logic matches real ATS checker behavior.
- Privacy-first approach is a strength because resume data is sensitive.
- Interview tracker makes the product more unique than a basic checker.
- UI is cleaner and easier to understand than the first version.

## Main Weaknesses

- The public URL/project name `write-a-code-for-my-website` feels like a test project, not a resume product.
- Product identity needs to be immediately clear: users should know this helps them get a job.
- The first impression needs more urgency and emotional clarity.
- Product modules should be separated clearly:
  - Resume Builder
  - ATS Checker
  - Interview Tracker
- The page needs stronger visual hierarchy, stronger CTA buttons, and a more product-like feel.
- The ATS score should feel more powerful with clearer section-wise scoring and visual feedback.

## Recommended Product Direction

Think of the product as:

```text
Create Resume -> Optimize Resume -> Track Interviews -> Get Job
```

The homepage should make two main user paths obvious:

- Create Resume
- Check ATS Score

## Branding Ideas

Possible names:

- ResumeIQ
- ResumeAI
- ResumeBoost
- HireReady
- ATS Checker Pro

Current preferred name: `ResumeIQ`.

## Next Action Plan

1. Rename or re-alias the Vercel URL to something resume-related.
2. Keep the headline direct: "Check if your resume passes ATS filters".
3. Make the two main CTAs visually dominant:
   - Create Resume
   - Check ATS Score
4. Continue improving the UI with icons, cards, spacing, and score visuals.
5. Improve the resume builder with templates and PDF export.
6. Add one strong "wow" element, such as:
   - Animated score bar
   - Bold rejection-risk insight
   - Highlighted missing keywords

## Product Mindset

Stop thinking of this as only a tool. Treat it as a product people can use to create, optimize, track, and improve their job search.
